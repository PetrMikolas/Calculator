# Kalkulačka

Webová kalkulačka na základní aritmetické operace. 

Naprogramováno v jazyce C# v prostředí Microsoft .NET 8 

- App/Client - Blazor Web App 
- API - gRPC
- Databáze - Dapper/Stored procedures
- Unit testy - NUnit
